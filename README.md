## Requisitos
- Visual Studio 2022
- .NET Framework 4.7.2 o superior
- Python 3.x
- Git
---
## Cómo ejecutar
1. Clonar el repositorio:
```bash
git clone https://github.com/TU_USUARIO/NOMBRE_DEL_REPO.git
```
2. Navegar a la carpeta del ejercicio que te interese.
3. Para proyectos en C#: abrir el archivo `.sln` con Visual Studio y presionar `F5`.
4. Para scripts en Python: ejecutar `python nombre_script.py` desde la terminal.
